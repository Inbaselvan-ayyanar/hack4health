# MediMesh: AI-Driven Medication Access and Adherence
A mobile-first web app for India's diverse population, featuring AI tutorials with Stable Diffusion illustrations, adherence circles, notifications, and accessibility (voice navigation, high-contrast UI).

## Setup
1. Clone: `git clone https://github.com/<username>/MediMesh-Hackathon.git`
2. Install: `npm install` (React, Tailwind via CDN for hackathon)
3. Run: Open `index.html` in a browser.

## Features
- Pharmacy Finder (PMBJP)
- Adherence Tracker and Circles
- AI Tutorials (text, audio, AI-generated images)
- AI Adherence Report
- Accessibility: TTS, high-contrast UI, visual cues
